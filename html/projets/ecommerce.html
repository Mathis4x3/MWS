Screen de l'écran d'accueil du site web

Ce projet nous a été confié début octobre 2024 et nous avions 1 mois pour rendre un site web E-commerce dynamique en PHP, lié à une base de données. Le but étant de donner aux futurs utilisateurs un interface simple, facile à utiliser et sécurisé.

Le site web devait contenir les fonctionnalités principales suivantes :

- La mise en place d'un panier pour les utilisateurs connectés et non connectés

- La gestion de différents types de produits

- La possibilité de commander les articles du panier

- Pouvoir supprimer/créer/modifier/visualiser les informations de chaque utilisateurs et produits

- Etre sécurisé et pouvoir différencier les utilisateurs des administrateurs

Screen de la page "panier" d'un utilisateur

J'ai pu lors de ce projet, participer à la gestion de la base de données du site web. En effet, avant de créer le site web, il a fallu créer le schéma relationnel de la base de données, ainsi pouvoir la lier au code PHP du site. Ce projet m'a permis d'en apprendre plus sur la sécurisation des données mais aussi des sites web de manière plus générale.
